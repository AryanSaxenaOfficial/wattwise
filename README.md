# WattWise Electricity Calculator
WattWise is a Public, Open-Source, Web-Page form software to calculate your electric bills.  
What started as a simple classroom-level exhibition project, is now beginning to feel like one of the most decorated calculators out there.
A baby software, still in its pre-alpha stage, it offers a huge deal more than any other tool. Just imagine not having to work in a totally scuffed environment. Imagine a software, not built for money, but for the simple love of coding.  
Hence I present to you - 
### WattWise
____

Built with basic languages and no obfuscation, its simple code allows any user to understand, replicate and add onto it.<sup>1</sup>
