The Explanation -

  1. You type in your values in the inputs
  2. Then the code multiplies each row's inputs
  3. Then all rows' worth of WattHours get added
  4. Then the code takes these WattHours and multiplies them one-after-the-another by the prices of Electricity in Aundh
  5. Then it returns the prices!
